# Simple Chat App (Socket.IO + TypeScript)

A very simple chat app built using Socket.IO and TypeScript. Running on a very basic server, fingers crossed it survives 😅. Multiple users can join, chat, and see who’s online .

## Installation

```bash
git clone https://github.com/sagarkemble/Simple-chat-app-using-socket.io
cd Simple-chat-app-using-socket.io
pnpm install
pnpm run dev
```
